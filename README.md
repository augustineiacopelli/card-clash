# Card Clash

A collection of classic card games, built as a single self-contained web page. No installs, no accounts, no build step — open `index.html` in any browser and play.

Built by Simon Iacopelli.

## Games

- **War** — flip against the House (or pass-and-play against a second player). Ties trigger a War, with escalating Double and Triple War banners for chained ties.
- **Go Fish** — collect books of four matching cards. Play against the House or pass-and-play with a friend, with a hidden-hand “pass the device” screen for two players.
- **Memory Match** — flip cards to find matching pairs. Choose how many pairs to play with, from a quick 4-pair round up to the full pack.
- **Solitaire** — classic one-player Klondike. Build the four foundations from Ace to King while clearing the tableau.

## Decks

Every game lets you choose between two decks before you start:

- **Bicycle** — the classic 52-card deck.
- **Wild Twists** — the same 52 cards plus four special WILD cards that bend the rules of whichever game you’re playing (instant wins and bonus steals in War, their own book in Go Fish, a matching pair in Memory Match, and a flexible fifth foundation in Solitaire).

## Running it

This is a static site — there’s nothing to build or install. Open `index.html` directly in a browser, or serve the folder with any static file host (including GitHub Pages).

## Tech

Vanilla HTML, CSS, and JavaScript in a single file. No frameworks, no dependencies beyond Google Fonts.
